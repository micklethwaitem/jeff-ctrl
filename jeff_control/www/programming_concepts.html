Programming Concepts

Variables

int a = 1;

char b = 'f';

Variables are place holders that store one piece of data each. They are given a name by the programmer so that each variable can be referred to easily. In the above example, a is the name the programmer gave to this variable and the number value of 1 has been stored inside it. When the program now finds the letter a, it will read the value 1.

Conditional Statements

if (a == b) {

  Do something;

} else {

  Do something else;

}

Conditional statements are used when a programmer wants to check a condition before deciding what the program does next. A light controlled by a computer would use a conditional statement. If the lightswitch is clicked on, then the light should turn on. If the lightswitch is off, the light should turn off. In the above example, the program asks if a is the same as b. If it is, it executes the code inside the first set of curly brackets. If a is not equal to b, it executes the code inside the second set of brackets.

Iterative Constructs

while (i == 0) {

  Do something;

}

Loops (also known as iterative constructs) repeat a block of code for a pre-specified number of times or unitl a condition has been fulfilled. The above example executes the block of code inside the curly brackets while i has the value '0' stored inside it. When that value changes, the program stops repeating that block of code.
